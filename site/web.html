Sviluppo Web per Bambini

Costruire un sito web è come costruire con i mattoncini: metti insieme tanti pezzi per creare qualcosa di nuovo.

  1. Apri un editor di testo semplice come Notepad (o un editor online).
  2. Scrivi questo codice di base:
<!DOCTYPE html>
<html>
<head>
    <title>Il mio primo sito</title>
</head>
<body>
    <h1>Ciao, Mondo!</h1>
</body>
</html>

Salva il file con estensione .html e aprilo con il tuo browser preferito.

Divertiti a cambiare i testi, i colori e le immagini per personalizzare il tuo sito!
