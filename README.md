# Tutorial AI e Web per Bambini

Questo repository contiene un semplice sito web pensato per aiutare i bambini di circa 10 anni a muovere i primi passi nell'uso dell'intelligenza artificiale e nello sviluppo di applicazioni web.

I file del sito si trovano nella cartella `site`:

- `index.html` – pagina principale con i collegamenti ai tutorial.
- `ai.html` – spiegazione dei concetti base di intelligenza artificiale.
- `web.html` – guida passo passo per creare la tua prima pagina web.

Apri `index.html` in un browser per esplorare i tutorial.
